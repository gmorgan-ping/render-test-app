## Quick Start

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Or start in production mode**:
   ```bash
   npm start
   ```

4. **Open your browser**:
   Visit `http://localhost:3000`

- **Backend**: Node.js, Express.js

- **Icons**: Font Awesome
- **Development**: Nodemon for auto-reload
